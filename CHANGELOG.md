# Changelog

All notable changes to `filament-reverb-doctor` will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-08

### Added
- Initial release
- Filament 3.x dashboard page for Reverb diagnostics
- Dashboard widget with quick health status
- 10 comprehensive diagnostic checks via laravel-reverb-doctor
- Configurable navigation (icon, group, label, sort order)
- Authorization support (permissions, gates, callbacks)
- Appearance customization (show/hide suggestions and details)
- Loading states with Filament notifications
- Dark mode support
- Laravel 10.x, 11.x, and 12.x support
